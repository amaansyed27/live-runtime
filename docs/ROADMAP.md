# Roadmap

## Phase 1: Working local runtime

- Tauri tray shell
- React dashboard
- Ollama chat streaming
- OS-native speech output
- Browser/WebView speech recognition fallback

## Phase 2: Native voice stack

- Native speech-to-text module per platform
- Global push-to-talk shortcut
- Optional wake-word process
- Audio device selection

## Phase 3: Runtime memory and tools

- Local encrypted profile store
- Tool permission model
- App and shell command skills
- Per-model system prompts

## Phase 4: Distribution

- Signed Windows installer
- Signed macOS DMG
- Linux AppImage/deb/rpm
- Update channel
